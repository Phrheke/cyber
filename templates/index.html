How to Use

This Implementation allows you to explore the five layers of the pyramid. Each layer represents a different concept:

  • Foundation - The base of the pyramid, representing the fundamental principles.
  • Core - The central layer, representing the heart of the Framework.
  • Adaptation - A layer focused on the ability to adapt.
  • Integration - Representing the integration of various systems.
  • Resilience - The final layer, symbolizing the Framework's strength and ability to endure.
Click on any layer to see more details about it. You can select a scenario and see each layers response for that case.
